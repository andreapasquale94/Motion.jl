# Motion.jl

_A Multibody Trajectory Design Toolkit_

[![CI](https://github.com/andreapasquale94/Motion.jl/actions/workflows/CI.yml/badge.svg?branch=master)](https://github.com/andreapasquale94/Motion.jl/actions/workflows/CI.yml)
[![Docs](https://img.shields.io/badge/docs-dev-blue.svg)](https://andreapasquale94.github.io/Motion.jl/dev)

**Motion.jl** is a fast, composable, and research-friendly Julia toolbox for simplified multi-body astrodynamics, focused on the usage semi-analytical models used in mission analysis (e.g. CR3BP, BCR4BP, ...).

## Installation

```julia
using Pkg
Pkg.add(url="https://github.com/andreapasquale94/Motion.jl")
```
